# Xs 编程语言
Xs 是一个专注于简单的开源跨平台编程语言。  

这门语言的设计目标是改进阅读与编写效率，降低语法负担，让使用者能够把真正的注意力放在解决问题上。  

因此我们舍弃了很多繁杂的特性，只保留最通用的功能，最终 Xs 只需极少的语法，就能优雅地表达逻辑。  
## 特性
- 精心设计的语法，易于编写和阅读。
- 规则明确并且统一，符合直觉。
- 目前支持 .NET 平台，借助 .NET 的框架和库资源，我们可以在非常广泛的场景下使用这门语言。
