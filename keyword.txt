//关键字
exp -> export 导出
imp -> import 导入

var -> variable 变量
invar -> invariable 不变量
const -> constant 常量

mtch -> match 匹配

lp -> loop 循环
brk -> break 跳出
ctu -> continue 继续

func -> function 函数
rcv -> receive 接收
rtn -> return 返回

strc -> structure 结构
actn -> action 行为
prot -> protocal 协议
slf -> self 自身

df -> defer 延迟执行
excp -> exception 异常监听
rep -> report 报告异常

usf -> unsafe 不安全区块声明，暂时只保留
